## 1.随机变量
这里有几个概念需要区分：随机试验、随机事件和随机变量。  
- 随机试验E：做实验，观察结果。
- 随机事件A：样本空间的**子集**。在某次试验中，如果这个子集中的元素出现，则称**事件发生**。
- 随机变量X=X(e)：定义在样本空间上的实值单值**函数**。 **函数！函数！函数！**
